## User Guide (Visual Studio Code)
The introduction video uses Visual Studio Code (VS Code) to run the code. It is strongly recommended to review and execute this project in VS Code.

### General Setup Steps:
1. **Download the Project:**
   - **Method A (ZIP):** Download the project files and extract the ZIP folder.
   - **Method B (Git Clone):** Open your terminal and run the following command to clone the repository:
     ```bash
     git clone https://github.com/Minicho8/COMP2090SEF_GROUP39.git
     ```
2. **Open in VS Code:** Open Visual Studio Code, click `File > Open Folder`, and select the project folder.
3. **Install Dependencies:** Open a new Terminal in VS Code (`Terminal > New Terminal`) and run the `pip install` command mentioned in the Prerequisites above.

### Running Task 1 (Restaurant Search Application)
Task 1 is an python application for searching, filtering, and locating restaurants near HKMU campuses.
1. Open up a new Terminal in VS Code.
2. Navigate to the Task 1 directory: 
   ```bash
   cd Task1
   ```
3. Run the main application program: 
   ```bash
   python main.py
   ```
   *(Alternatively, open `Task1/main.py` in the VS Code editor and press `Ctrl + F5` or click "Run Without Debugging".)*

### Running Task 2 (Sorting & Algorithms)
Task 2 contains algorithm implementations (such as Radix Sort).
1. Open up a new Terminal in VS Code.
2. Navigate to the Task 2 directory:
   ```bash
   cd Task2
   ```
3. Run the desired algorithm script. For example, to run the Radix Sort:
   ```bash
   python Radix_Sort/radix_sort.py
   ```
   *(Alternatively, open the specific Task 2 script in the editor and press `Ctrl + F5`.)*

---
*Note: If you run the code outside of VS Code (e.g., standard Command Prompt or Terminal), assure you navigate directly into the `Task1` or `Task2` directories before executing the `python` commands to avoid relative file path errors (such as missing CSV files).*
